# Medieval War

中世纪战争自动对战小游戏。购买兵种、运营经济、摧毁敌方城堡！

[🎮 Play Now](https://kriines.github.io/medieval-war/)

## How to Play

- 点击底部兵种按钮购买单位（或键盘 1-6）
- 单位自动向敌方城堡进军，途中遇敌自动战斗
- 挖矿者(⛏)产生金币收入
- 摧毁敌方城堡获胜
